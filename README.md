# Threat Log Monitor

## Overview

Threat Log Monitor is a Python-based cybersecurity automation project that analyzes log files and detects suspicious activities.

## Features

- Failed Login Detection
- Brute Force Attack Detection
- Port Scan Detection
- Alert Generation
- Timestamped Alerts



## How to Run

```bash
python threat_monitor.py
```

## Sample Output

2026-06-24 17:24:08 [CRITICAL] Possible Brute Force Attack from 192.168.1.10 (3 failed logins)

2026-06-24 17:24:08 [WARNING] Possible Port Scan from 192.168.1.15 (2 port scan events)
